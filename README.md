This project implements an intelligent Multi-Tool AI Agent capable of interacting with medical datasets and the web. Built using **LangChain**, **Groq (Llama-3)**, and **Tavily API**, the agent can intelligently route user queries to either a SQL database (for statistical data) or a Web Search tool (for general medical knowledge).


## ğŸš€ Features

* **Database Interaction:** Converts CSV datasets into SQLite databases and executes SQL queries to answer questions about Heart Disease, Diabetes, and Cancer.
* **Web Search Capability:** Uses Tavily Search API to fetch real-time medical information (symptoms, cures, definitions).
* **Intelligent Routing:** The agent automatically decides whether to query the database or search the web based on the user's question.
* **Self-Correction:** If a generated SQL query fails, the agent attempts to fix the query logic automatically.

##  âš™ï¸ Installation & Setup
Follow these steps to set up the project locally in VS Code.

1. Create and Activate Virtual Environment
It is strictly recommended to use a virtual environment to avoid version conflicts.




# Create virtual environment
```
python -m venv venv
```
# Activate the environment (Windows)
```
.\venv\Scripts\activate
```
# Activate the environment (Mac/Linux)
```
source venv/bin/activate
```
2. Install Dependencies
Run the following commands to install the necessary libraries. If you face version issues, use the force-reinstall command.


# Standard installation
```
pip install pandas langchain langchain-groq langchain-community tavily-python sqlalchemy python-dotenv
```
# OR (If you face version errors, use this nuclear method)
```
pip install --upgrade --force-reinstall --no-cache-dir langchain langchain-community langchain-core langchain-groq tavily-python
```
3. Configure API Keys
Create a file named .env in the root directory and add your API keys:
```
GROQ_API_KEY=your_groq_api_key_here
TAVILY_API_KEY=your_tavily_api_key_here
```
4. Initialize Databases
Run the database creation script to convert the CSV files into SQLite databases:
```
python create_db.py
```
You should see a message confirming that heart_disease.db, cancer.db, and diabetes.db have been created in the databases/ folder.
